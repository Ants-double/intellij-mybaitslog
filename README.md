# TreeInfotip

### TreeInfotip能做什么？

> TreeInfotip是基于IntelliJ 开发的项目目录自定义备注显示，主要通过自定义XML来生成项目目录树备注。
---
### 使用环境
`IntelliJ IDEA Ultimate版（172+）`

`WebStrom（172+）`
